# 🚀 MobilityLakehouse

## 📌 Overview
MobilityLakehouse is a data engineering project built using Databricks, PySpark, and Delta Lake.  
It processes mobility-related datasets such as customers, drivers, trips, payments, and locations using a multi-layered architecture (Bronze, Silver, Gold).

---

## 🏗 Architecture
- **Source:** CSV files (Databricks Volumes)
- **Processing:** PySpark Structured Streaming
- **Storage:** Delta Lake (Lakehouse architecture)

---

## 📂 Datasets
- Customers
- Drivers
- Trips
- Payments
- Locations
- Vehicles

---

## ⚙️ Technologies Used
- Databricks
- PySpark
- Delta Lake
- Structured Streaming
